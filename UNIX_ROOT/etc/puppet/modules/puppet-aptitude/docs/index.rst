puppet-aptitude
===============

puppet-aptitude is a reusable `puppet <http://www.puppetlabs.com/>`_ module for
`ubuntu <http://www.ubuntu.com/>`_ servers that:

* Installs ``aptitude`` and ensures it is kept up-to-date.
* Updates the ``aptitude`` package list, so that you can install any version of
  any package.
* Optionally allows you to keep all packages up-to-date (system-wide).

Using puppet-aptitude is easy, and takes about 2 minutes to get up and running.
